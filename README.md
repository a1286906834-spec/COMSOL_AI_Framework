# COMSOL AI Framework v0.5 - Skill Optimized

This package keeps v0.5 scope and does not add v0.6 features.

## What changed

The original v0.5 was an engineering + multiphysics skeleton.

This optimized version adds operational skills under:

```text
framework/skills/
```

## Core Skills

- Chief multiphysics engineering
- CFD chief engineering
- Multiphysics coupling
- Conjugate heat transfer
- Fluid-structure interaction
- Electromagnetic heating
- Solver diagnosis
- Numerical stability
- Validation review
- Python mph realtime control

## Use

Place the project in your Codex workspace.

Run in Codex:

```text
/selfcheck
```

Expected:

```text
COMSOL-AI-FRAMEWORK-v0.5-SKILL-OPTIMIZED
```

## Principle

Do not automate COMSOL before engineering review.

Do not overwrite `.mph` files without explicit approval.

Do not guess missing physics, geometry, materials, or boundary conditions.
